## Usage Rules

- Product thinking, design notes, safety decisions, learning content plans, and iteration logs live in Obsidian.
- Source code, API implementation, tests, and runnable assets stay in this project.
- Do not store Hermes keys, model keys, private child chat logs, or sensitive credentials in Obsidian notes.
- Use Obsidian wikilinks for related notes, for example `[[产品定位与范围]]`.
